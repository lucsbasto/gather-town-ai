# Interface Design System - Gather Town AI

## Direction
- **Feel:** Precision & Density
- **Best for:** Developer tools, game interfaces
- **Depth:** Borders-only (clean, technical)

## Tokens

### Spacing
- Base: 4px
- Scale: 4, 8, 12, 16, 24, 32

### Colors (Game/Dark Theme)
- Background: #1a1a2e
- Surface: #16213e
- Primary: #0f3460
- Accent: #e94560
- Text: #ffffff
- Text Secondary: #a0a0a0

### Typography
- Font: System (sans-serif)
- Size: 14px base, 12px small, 16px large

## Patterns

### Button Primary
- Height: 36px
- Padding: 12px 16px
- Radius: 6px
- Background: #0f3460
- Hover: #e94560

### Card
- Border: 1px solid rgba(255,255,255,0.1)
- Padding: 16px
- Radius: 8px
- Background: #16213e

### Input
- Height: 36px
- Padding: 8px 12px
- Border: 1px solid rgba(255,255,255,0.2)
- Background: #1a1a2e

## UI Components Needed
- Player avatar (32x32 grid)
- Movement joystick/controls
- Proximity indicator
- Video/audio call UI
- Room list
- Chat overlay
- Settings modal
